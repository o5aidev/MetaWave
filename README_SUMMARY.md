# MetaWave - プロジェクト概要

## 🎯 MetaWaveとは？

**MetaWave**は、メタ認知力を向上させるためのiOSアプリです。音声入力、感情分析、パターン分析、予測機能を備えた個人のメタ認知パートナーです。

## 🚀 使い方

### 実行方法
1. XcodeでMetaWave.xcodeprojを開く
2. ⌘R で実行
3. ⌘U でテスト実行

### 主な機能
- **Notesタブ**: ノート一覧と作成
- **分析タブ**: パターン分析、予測分析
- **設定タブ**: アプリ設定
